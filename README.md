# 🚀 Flyconsole V2.0 - Advanced Security Suite

**Flyconsole** is an all-in-one terminal-based framework designed for network scanning, reconnaissance, and security auditing. It simplifies complex security tasks into an easy-to-use interactive menu.

## 🛠️ Core Features
- 🔍 **Comprehensive Scanning:** Advanced SYN/TCP port scanning.
- 🛠️ **Service Detection:** Identifies running services and their versions.
- 💻 **OS Fingerprinting:** Predicts the target's operating system.
- 🪤 **Deceptive Honeypots:** Integrated HTTP/FTP honeypots for intruder tracking.
- 📝 **Auto-Reporting:** Generates clean HTML reports of scan results.
- 🌐 **Web Security:** Basic analyzer for headers and sensitive paths.
